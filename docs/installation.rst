============
Installation
============


Stable release
--------------

To install Network Heat Diffusion, run this command in your terminal:

.. code-block:: console

    $ pip install cddiffusion

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Network Heat Diffusion can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/idekerlab/cddiffusion

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/idekerlab/cddiffusion/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/idekerlab/cddiffusion
.. _tarball: https://github.com/idekerlab/cddiffusion/tarball/master
